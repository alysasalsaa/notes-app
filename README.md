# notes-app

A Markdown-enabled notes app with local storage persistence.

## Features

- [x] Create, edit, and delete notes
- [x] Notes persist in localStorage across browser sessions
- [x] **Bonus:** Write notes in Markdown, rendered as HTML on preview
- [x] **Bonus:** Creation/update date shown per note
- [x] Search notes by title or content
- [x] Split view (editor + preview side by side)
- [x] Auto-save while typing

## Tech Stack

- HTML
- CSS
- JavaScript (vanilla)
- [marked.js](https://github.com/markedjs/marked) — Markdown parser (via CDN)

## How to run

Open `index.html` in your browser — no build tools needed.
